Para crear o actualizar tus productos, puedes subir un archivo en formato csv. Es importante que consideres las siguientes indicaciones con el contenido de tu archivo para que la actualización sea efectiva

Columna Formato
Id Con este campo tus productos serán registrados en la base de datos. Si el Id existe, el producto será actualizado Este campo es obligatorio
Precio Con este campo tus productos mostrarán sus precios.Es importante que subas este dato con separación de millares con COMMA (,) y decimales con PUNTO (.) y sin signo de moneda ($,€,¥...) Este campo es obligatorio
Referencia Ccorresponde a una nombre, título o referencia del producto, es un texto breve como "Pizza 2 ingredientes"
Descripción Es la descripción de producto, puede ser un texto exteno pero en una sola celda
Categorias Corresponde a una lista de categorías separadas por comas, con este campo tus productos serán organizados en categorías. Las categorías que no existan en la base de datos, serán creadas. Ejemplo categoria, categoria, cate...
Subcategoria Corresponde a una categoría seguido de una lista de subcetegorías. El formato de tu celda debe ser Categoria:Subcategira, subcategoria, sub...
On Stock Corresponde a una celda que especifique si el producto está en existencia o no
Cantidad de Stock Corresponde a la cantidad de existencia del producto. Es importante que subas este dato con separación de millares con COMMA (,) y decimales con PUNTO (.)
Imagen Corresponde a un link de imagen donde tienes alojadas tus imágenes. No admite archivos de imágenes
